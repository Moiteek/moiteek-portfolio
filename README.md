## 🚀 Deployment

### **GitHub Pages**
1. Create a new repository named `portfolio`
2. Upload the entire `portfolio` folder
3. Go to Settings → Pages
4. Select "Deploy from a branch" → "main" → "/ (root)"
5. Your site will be live at `https://[username].github.io/portfolio`

### **Netlify**
1. Drag and drop the `portfolio` folder to [netlify.com](https://netlify.com)
2. Wait for deployment (2-3 minutes)
3. Your site will be live with a random URL

### **Vercel**
1. Connect your GitHub repository
2. Import the `portfolio` folder
3. Deploy automatically
